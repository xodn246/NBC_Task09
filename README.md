# NBC Task09 - 숫자 야구 게임

## 과제 소개

Unreal Engine과 C++를 활용하여 구현한 멀티플레이어 숫자 야구 게임입니다.
실시간 채팅 방식으로 진행되며, 서버에 접속한 플레이어 중
먼저 정답을 맞추는 플레이어가 승리합니다.

---

## 게임 규칙

- 게임 시작 시 중복 없는 3자리 숫자가 정답으로 설정됩니다.
- 서버에 접속한 순서대로 플레이어가 추가되며, 인원 제한은 없습니다.
- 플레이어들은 번갈아가며 입력하는 방식이 아닌, 실시간 채팅 방식으로 자유롭게 숫자를 입력합니다.
- 각 플레이어는 최대 3번의 입력 기회를 가집니다.
- 입력한 숫자에 대해 Strike / Ball / Out 결과를 제공합니다.
- 입력 기회 내에 정답을 맞춘 플레이어가 승리합니다.
- 모든 플레이어가 입력 기회를 소진하면 Draw(무승부)로 게임이 종료됩니다.

---

## 구현 과정

### 1. 프로젝트 세팅
- Unreal Engine C++ 프로젝트 생성
- GameMode 및 기본 클래스 구조 설계

### 2. 게임 로직 구현
- 중복 없는 3자리 랜덤 숫자 생성 로직 작성
- 입력값과 정답을 비교하여 Strike / Ball 계산
- 서버 접속 순서대로 플레이어를 동적으로 추가하는 구조 설계
- 실시간 채팅 방식의 입력 처리 구현
- 플레이어별 남은 입력 기회(3회) 관리
- 정답 맞춤 시 해당 플레이어 승리 처리
- 전원 기회 소진 시 Draw 처리

### 3. 빌드 및 테스트
- 언리얼 에디터에서 실행 및 게임 흐름 검증
